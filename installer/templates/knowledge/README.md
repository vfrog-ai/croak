# CROAK Knowledge Base

This directory contains curated knowledge that CROAK agents use to provide guidance.

## Structure

```
knowledge/
├── architectures/     # Model architecture guides
├── training/          # Training best practices
├── evaluation/        # Evaluation methodologies
├── deployment/        # Deployment guides
├── troubleshooting/   # Common issues and solutions
└── gpu-setup/         # GPU provisioning guides
```

## Adding Knowledge

Knowledge files are markdown documents loaded by agents based on context.

Each file should include:
- Clear headings for sections
- Practical examples
- Links to external resources
